# 📌 Git Commit Message 전략

## Type 종류

| Type         | 설명                                   |
| ------------ | -------------------------------------- |
| **feat**     | 새로운 기능 추가                       |
| **fix**      | 버그 수정                              |
| **docs**     | 문서 수정 (README, 주석 등)            |
| **style**    | 코드 스타일 변경 (포맷팅, 세미콜론 등) |
| **refactor** | 리팩토링 (동작 변화 없음)              |
| **test**     | 테스트 코드 추가/수정                  |
| **chore**    | 빌드, 패키지 매니저, 잡일 등           |
| **build**    | 빌드 관련 설정 변경 (CI/CD 포함)       |
